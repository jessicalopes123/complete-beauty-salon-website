# Complete Beauty Salon Website

Site responsivo para salão de beleza desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Página inicial moderna
- Apresentação dos serviços
- Vídeo institucional
- Integração com Instagram
- Layout responsivo
- Design moderno e elegante

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo
Praticar desenvolvimento frontend criando um site moderno para salão de beleza com foco em usabilidade e responsividade.
